오디언스 조합 가이드

1. 세그먼트 배치하기

- 만들어진 세그먼트를 조합 영역으로 드래그하여 배치합니다.

2. 세그먼트 결합하기

세그먼트는 높이를 3만큼 결합을 구성할 수 있습니다.

- 세그먼트를 2개 이상 배치합니다.

- 하나의 세그먼트를 드래그하여 다른 세그먼트 근처로 가져갑니다.
- 근처로 드래그 하면 결합 대상은 점선으로 표시됩니다.

- 결합 된 세그먼트를 확인합니다.
- 결합 된 세그먼트는 기본적으로 교집합 설정을 가지게 됩니다.

3. 집합 설정

- 결합된 세그먼트의 각 면을 우클릭 합니다.
- 우클릭을 누르면 각 면이 변경되는 것을 볼 수 있습니다.
- 교집합(∩), 합집합(∪), 차집합(-)을 사용 가능합니다.
- 대칭차집합(△)의 경우 지원하지 않음으로 교집합(∩)으로 자동 변경 됩니다.
- 모두 선택하지 않은 경우는 합집합(∪)으로 자동 변경 됩니다.

4. 집합 해제 및 세그먼트 삭제

- 결합된 세그먼트를 우 더블 클릭하면 단일 세그먼트로 해제가 가능합니다.

- 단일 세그먼트를 우 더블 클릭하면 삭제가 가능합니다.
